## 📖 About

**ReshavOS** is a browser-based desktop experience that reimagines the traditional portfolio as a fully interactive operating system.

Instead of browsing static web pages, visitors can navigate a virtual desktop, launch applications, explore projects, interact with widgets, use a terminal, listen to music, switch themes, and experience a desktop-inspired interface—all within the browser.

The project was built from scratch using **HTML, CSS, and Vanilla JavaScript**, focusing on creativity, user experience, and interactive design.

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/koley-dev-08/personal-os.git
```

Open the project folder:

```bash
cd personal-os
```

Launch `index.html` in your browser or use a local development server.
